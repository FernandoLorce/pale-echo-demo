苍白回声 (Pale Echo)
一个基于浏览器的交互式叙事探案游戏

关于游戏
《苍白回声》是一款第一人称的侦探推理游戏，您将扮演俄勒冈州警探凯特·帕克斯。游戏模拟了一个完整的警用案件管理系统，您需要通过浏览卷宗、检查证物、分析数字取证信息以及进行关键词搜索，来揭开一桩发生在废弃天文台的离奇死亡案件背后的真相。

游戏的核心驱动力是探索与发现。随着您深入调查，一个跨越百年的家族秘密、一系列看似毫无关联的死亡事件以及一首神秘诗篇将逐渐浮出水面。您不仅是在调查一起案件，更是在解读一个关于献祭、传承与选择的黑暗寓言。

游戏特色：

沉浸式界面：还原警方案件管理系统的操作界面，包括任务栏、侧边栏和进度追踪。

多维度调查：整合了现场照片、法医报告、死者信息、证据清单、数字取证工作站、模拟搜索引擎和邮件系统等多种信息来源。

线索拼图：您需要从碎片化的信息中建立关联。

叙事驱动：剧情通过邮件、备忘录、社交媒体帖子和审问记录等多种叙事载体展开，体验层层递进的悬疑氛围。

动态反馈：您的调查进度会解锁新的线索、邮件和剧情事件，推动故事走向结局。

故事背景
2026年3月19日，清晨8时14分。奥克湾警局接到一通报警电话——废弃多年的学院天文台内发现一具尸体。

死者是27岁的本地居民诺亚·布伦南。死因初检判定为咸水溺亡——而发现尸体的房间干燥、无水源。

尸体衣物整洁、干燥、无褶皱。门锁内侧无破坏痕迹。地面画有一圈不属于任何已知文字体系的符号。

这不是一具被丢弃的尸体。它是一段被放置在这里的信息。

您，凯特·帕克斯探员，将接手这个代号为“苍白回声”的谜案。随着调查的深入，您将发现诺亚的死只是冰山一角，它指向了一个始于1911年、由灯塔看守人之女所著的黑暗诗篇.

主要角色

马库斯·霍尔特 (Marcus Holt) - OSP探长，您的上司，可以提供权限和资源。

卡拉·门德斯 (Cara Mendez) - OSP数字取证专家，负责分析和提取电子设备中的信息。

玩法与操作指南
游戏完全通过点击和键盘输入进行。

1. 主界面
顶部任务栏：显示案件名称、状态和当前登录的探员信息。

左侧导航栏：您的主要调查工具，点击图标可在不同功能模块间切换：

案件主控台: 案件概览和快速入口。

现场照片: 查看案发现场的图像记录。

初步报告: 阅读法医、巡警和CSI的官方报告。

死者信息: 查看受害者诺亚·布伦南的个人资料。

证据清单: 浏览所有已收集的物证。

取证工作站: 分析数字证据，如手机镜像和笔记本电脑数据。

搜索终端: 核心玩法。输入关键词进行搜索，发现隐藏的档案和线索。

邮件系统: 接收和阅读来自其他角色的邮件，推进剧情。

线索白板: 整理和串联所有关键线索，完成核心谜题（问卷）。

2. 核心机制：搜索与线索发现
在“搜索终端”的输入框中输入关键词（如 Andrew Farrow）并按回车键。

系统会返回相关的档案条目。点击条目可以查看详细信息。

某些敏感档案需要您从别处获取授权密码后才能解锁。

搜索特定关键词是触发新邮件和后续剧情的关键。

3. 进度与反馈
页面底部的进度条会大致反映您的调查进展。

新邮件到达时，屏幕右上角会弹出“Toast”通知，点击即可阅读。

关键的剧情转折会通过全屏动画或模态框呈现。
